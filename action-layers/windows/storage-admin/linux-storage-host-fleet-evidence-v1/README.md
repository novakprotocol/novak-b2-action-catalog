# linux-storage-host-fleet-evidence-v1

Safe vendor storage operations evidence layer.

This layer performs no remediation, no mutation, no target scan, and stores no credentials or raw target identifiers.

Linux storage host fleet evidence for multipath, SCSI, LVM, filesystems, NFS, CIFS, iSCSI, FC, fstab, autofs, udev, logs, and Commvault agents.
