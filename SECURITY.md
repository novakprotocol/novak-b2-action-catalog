## Do not submit

Do not submit credentials, tokens, IP addresses, hostnames, usernames, raw event logs, workplace target lists, private evidence, customer data, or production details through GitHub.

## Scope

This repository is a public source-available action catalog. It does not approve production deployment, admin repair, workplace targeting, or automatic remediation.

## Reporting

Public GitHub issues and discussions are currently closed. Security review and intake will be defined later if the project owner opens a formal process.

## Operator responsibility

Anyone using or adapting this repository is responsible for validating safety, privacy, policy, and operational fit in their own environment.
